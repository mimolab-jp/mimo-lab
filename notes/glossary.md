# 🌸みもLAB 用語辞典

# 📖 Glossary（用語辞典）

みもLABで学んだ用語をまとめた辞典。
アルファベット順で管理する。

---

# A

## Auto Rename Tag

VS Codeの拡張機能。

開始タグを書き換えると、対応する閉じタグも自動で変更してくれる。

例

```html
<div>
```

↓

```html
<section>
```

すると

```html
</div>
```

↓

```html
</section>
```

へ自動で変更される。


## Agent

CursorのAI実行モード。

指示内容に応じて

複数ファイルを自動で編集できる。

---

## Ask

Cursorの相談モード。

コードは変更せず、

質問や設計相談を行うモード。

---

# C

## class

HTML要素につける「名前」。

HTML

```html
<div class="card">
```

CSS

```css
.card
```

このようにCSSから指定するときに使用する。

---

## classList

【読み】クラスリスト

class（ラベル）一覧。

JavaScriptからCSSクラスを付けたり外したりする仕組み。

---

## contain

【読み】コンテイン

意味：収める・含む・中に入れる。

```css
object-fit: contain;
```

画像全体が枠の中に収まるように表示する。

画像は切れないが、余白ができる場合がある。

---

## const

【読み】コンスト

正式名称：constant

意味：変更しない箱（変数）を作る。

```javascript
const name = "山さわ";
```

---

## cover

【読み】カバー

意味：覆う・覆い尽くす。

```css
object-fit: cover;
```

画像で表示枠をいっぱいに覆う。

画像の一部は切れるが、余白はできない。

---

## CRUD

【読み】クラッド

Create（追加）
Read（表示）
Update（更新）
Delete（削除）

アプリ開発の基本となる4つの機能。

---

## Cursor

AIを搭載したコードエディタ。

Visual Studio Codeをベースに開発されており、

AIチャット・Agent機能を利用できる。

---

# D

## Default Style

【読み】デフォルトスタイル

ブラウザが最初から持っている見た目。

CSSを書かなくても適用される初期デザイン。

---

## display

要素の表示方法を指定するCSSプロパティ。

よく使う値

- display: none;
- display: block;
- display: flex;

---

# E

## event.stopPropagation()

【読み】イベント・ストップ・プロパゲーション

クリックなどのイベントを親要素へ伝えない仕組み。

---

## Extensions（拡張機能）

VS Codeにあとから追加できる機能。

スマホでいう「アプリ」のようなもの。

---

# F

## filter()

【読み】フィルター

条件に合うデータだけ残して新しい配列を作る。

英語の意味は「ふるいにかける」。

---

## function

【読み】ファンクション

意味：機能。

処理をひとまとめにしたもの。

必要になった時だけ呼び出せる。

---

# G

## Google Maps Search URL

Google Mapで検索を開くURL。

店名や住所を渡すだけで検索結果を表示できる。

API登録なしで利用できる。


---


# H

## HTML

Webページの骨組みを作る言語。

タイトル・文章・画像・ボタンなど、
ページの内容そのものを書く。

---

# I

## includes()

【読み】インクルーズ

意味：含まれているか調べる。

```javascript
text.includes("山")
```

のように使う。

---

## innerHTML

【読み】インナーエイチティーエムエル

inner = 内側

HTML = HTML

HTMLの中身を書き換える仕組み。

---

# J

## JavaScript

Webページに「動き」を付けるプログラミング言語。

例

- ボタンを押した時の処理
- 画面切り替え
- 検索
- データ保存
- アニメーション

---

# L

## Live Server

VS Codeの拡張機能。

保存（Ctrl + S）するとブラウザが自動更新される。

---

## localhost

自分のPCだけで動くWebサーバー。

Live Serverによって作られる。

インターネット上には公開されない。

---

## localeCompare()

【読み】ロケールコンペア

文字列を言語順で比較する。

日本語の並び替えにも対応する。

---

## LocalStorage

【読み】ローカルストレージ

ブラウザ内にある小さな保存領域。

同じブラウザ・同じPCなら、
ブラウザを閉じてもデータを保持できる。

---

# O

## object-fit

画像を表示枠へどう収めるかを指定するCSSプロパティ。

代表的な値

- cover
- contain

---

## oninput

【読み】オンインプット

入力内容が変わるたびに実行されるイベント。

---

# P

## push()

【読み】プッシュ

意味：押し込む。

配列の最後へ新しいデータを追加する。

---

# R

## RAM

【読み】ラム

Random Access Memory

プログラム実行中の作業スペース。

保存先ではなく「作業机」のような存在。

電源を切る・ページを閉じると内容は消える。

---

## render

【読み】レンダー

意味：描画する。

画面を最新状態に描き直すこと。

ゲームのレンダリングと同じ語源。

---

## Responsive Design

【読み】レスポンシブデザイン

業界では「レスポンシブ」と略される。

画面サイズに応じてレイアウトや大きさを自動調整するデザイン手法。

---

# S

## show

【読み】ショー

意味：表示する。

showRamen() は
「ラーメンを表示する機能」という意味。

---

## sort()

【読み】ソート

意味：並び替えを行う。

---

## Syntax Highlight（シンタックスハイライト）

コードを種類ごとに色分け表示する機能。

例

- HTMLタグ
- CSSプロパティ
- 文字列
- 色コード

---

# V

## Visual Studio Code（VS Code）

現在最も利用者が多いコードエディタの一つ。

HTML・CSS・JavaScriptなど様々な言語に対応している。

拡張機能を追加することで、自分専用の開発環境を作れる。

---

# 記号

## .

【読み】ドット

意味

「○○の中の」

例

```javascript
document.getElementById
```

↓

document の中の getElementById

---

## =

【読み】イコール

意味

代入する。

右側の値を左側へ入れる。

---

## ()

【読み】丸括弧（まるかっこ）

意味

機能を呼び出す。

または、

機能へ情報（引数）を渡す。

例

```javascript
showRamen("山さわ")
```

---

## ""

【読み】ダブルクォーテーション

意味

文字列を表す。

---

## ''

【読み】シングルクォーテーション

意味

文字列を表す。

基本的な役割はダブルクォーテーションと同じ。

---

## ` `

【読み】バッククォート

意味

複数行の文字列を書ける。

`${}` を利用できる。

---

## ${}

【読み】ドル・波括弧

意味

変数の中身を取り出して文字列へ埋め込む。

例

```javascript
const name = "山さわ";

`こんにちは ${name}`
```

↓

こんにちは 山さわ
